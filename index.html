Regina's Rentables

A Web application for connecting with other users to find and rent out equipment in Regina

Technology

Regina's Rentables is built with the Django framework and Bootstrap. This project involved all steps of the SDLC, from requirements to implementation in an Agile environment. Forest Fires and Clustering

An unsupervised machine learning model that finds wildfire hotspots in California.

Technology

Created with Python 3.7, extracts California wildfire data points from an sqlite database on kaggle.com, selecting fires that burnt more than 100 acres from 2010-2015. Applies the k-means clustering algorithm from the scikit learn library to create a model with optimal locations for heavy firefighting equipment. Uses this model in a small fire response application. Maps the results on auto-generated HTML files with Folium (leaflet.js).

Technology
